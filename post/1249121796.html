HTML速成

HTML，超文本标记语言，并非编程语言，而是使用通常成对的标签来描述网页内容，可以嵌套描述。因为大部分情况下是针对一个模板进行修改或者优化，所以学习目标仅仅是看得懂，使用时查阅即可（本文附录）。

大部分前端代码可以用GPT生成，我们知道基本语法即可。

实例：

HTML标签

HTML元素始于开始标签，终于结束标签，部分元素不需要结束标签，比如换行<br>，但处于长远考虑，建议都加上结束标签</br>。

此外，HTML大小写不敏感，建议使用小写字母。常用的基础标签：

标签 描述
<h1>-<h6> 标题，h1最大
<hr> 创建水平线，用于分割内容
<p> 段落
<br> 换行

HTML属性

属性以键值对方式写在标签（一般是开始标签）内，以空格隔开，实例：

通过属性可以引入样式文件（css）和Js操作，常见属性：

属性 描述
class 为html元素定义一个或多个类名(类名从样式文件引入)
id 定义元素的唯一id
style 规定元素的行内样式（inline style）
title 描述了元素的额外信息 (作为工具条使用)
href <a>标签内指定链接目标
src <img>和<script>标签内指定资源路径
alt <img> 中提供图片替代文本
title 元素的额外信息，通常在鼠标悬停时显示
name <input>, <form>, <select> 等表单元素中使用，定义元素的名称
value 定义表单元素的值
target <a> 标签中指定链接的打开方式，_blank是在新标签中打开
type 指定表单元素的类型
placeholder 为 <input> 和 <textarea> 提供一个占位符文本
download <a>标签中下载文件

HTML基础用法

简单介绍一下一些基础用法。

头部标签

标签 示例 描述
<title> - 浏览器工具栏的标题
<style> <style type="text/css"> body { background-color:yellow; } p { color:blue }</style> 样式文件引用地址，也可以直接添加样式
<meta> <meta name="keywords" content="HTML教程">
<meta name="auther" content="作者名字">
<meta http-equiv="refresh" content="30">		 元数据，会被解析（最后一个是刷新）
<link> <link rel="stylesheet" type="text/css" href="mystyle.css"> 文档与外部资源关系，链接
<script> - 脚本文件
<noscript> - 无脚本文件
<base> <base href="http://www.baidu.com" target="_blank"> 默认的链接地址/链接目标

区块与分布

最核心的设计之一，通过<div>和<span>将元素进行组合，结合css进行区块的样式设计，推荐使用区块进行HTMl布局。

标签 描述
<div> 块级元素，它可用于组合其他 HTML 元素的容器，没有特定含义
结合css对大的区块进行样式设计
<span> 内联元素（行内），作为文本容器

表格与列表

表单输入

很常见的用法，通过表单收集用户的输入信息，涉及单选框、复选框等。

标签 描述
<form> 定义供用户输入的表单
<input> 定义输入域
<textarea> 定义文本域 (一个多行的输入控件)
<label> 定义了<input>元素的标签，一般为输入标题
<fieldset> 定义了一组相关的表单元素，并使用外框包含起来
<legend> 定义了<fieldset>元素的标题
<select> 定义了下拉选项列表
<optgroup> 定义选项组
<option> 定义下拉列表中的选项
<button> 定义一个点击按钮
<datalist>(new) 指定一个预先定义的输入控件选项列表
<keygen>(new) 定义了表单的密钥对生成器字段
<output>(new) 定义一个计算结果

盒子模型
