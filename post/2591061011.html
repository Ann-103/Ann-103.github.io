Vue3学习笔记

学习一下Vue3技术，仅记录一些核心知识，更多内容可参考官方文档：

Vue.js - 渐进式 JavaScript 框架 | Vue.js

Vue是用于构建用户页面的JavaScript 框架。它基于HTML、CSS 和 JavaScript 构建，并提供了一套声明式的、组件化的编程模型。

  • 减少了DOM操作，使开发者更加专注于数据逻辑，从而将数据和页面的逻辑分离。
  • 组件化开发，更适用于大型项目开发。

DOM操作：使用JavaScript通过文档对象模型（DOM）来操作网页的HTML元素和内容。DOM是一个树状结构，用于表示网页的结构和内容，并通过Javascript进行修改网页的元素和属性。

Vue的两个核心功能：

  • 声明式渲染：基于标准 HTML 拓展了一套模板语法，可以声明式地描述最终输出的 HTML 和 JavaScript 状态之间的关系。
  • 响应性：Vue 会自动跟踪 JavaScript 状态并在其发生变化时响应式地更新 DOM。

基础知识

单文件组件

单文件组件，也就是*.vue文件，会将一个组件的逻辑 (JavaScript)，模板 (HTML) 和样式 (CSS) 封装在同一个文件里。

API 风格

选项式 API (Options API)

以用包含多个选项的对象来描述组件的逻辑，例如 data、methods 和 mounted。选项所定义的属性都会暴露在函数内部的 this 上，它会指向当前的组件实例。

组合式 API (Composition API)

使用导入的 API 函数来描述组件逻辑，通常会与<script setup> 搭配使用。这个 setup attribute 是一个标识，告诉 Vue 需要在编译时进行一些处理,比如，<script setup> 中的导入和顶层变量/函数都能够在模板中直接使用。

我们之后的学习以组合式API为例。

组合式 API 的核心思想是直接在函数作用域内定义响应式状态变量，并将从多个函数中得到的状态组合起来处理复杂问题。

模板语法

在底层机制中，Vue 会将模板编译成高度优化的 JavaScript 代码。

指令由 v- 作为前缀，表明是一些由 Vue 提供的特殊 attribute，它们将为渲染的 DOM 应用特殊的响应式行为。这里我们做的事情简单来说就是：在当前组件实例上，将此元素的 innerHTML 与 rawHtml 属性保持同步。

每个绑定仅支持单一表达式，也就是一段能够被求值的 JavaScript 代码。

符号 解释 示例
{{}} 文本插值 <span>Message: {{ msg }}</span>
v-html 插入 HTML <p>Using v-html directive: <span v-html="rawHtml"></span></p>
v-bind 绑定attribute <div v-bind:id="dynamicId"></div>or<div :id="dynamicId"></div>
v-on 监听一个事件 <a v-on:click="doSomething"> ... </a>or <a @click="doSomething"> ... </a>
v-if 基于表达式 seen 的值的真假来移除/插入该 <p> 元素 <p v-if="seen">Now you see me</p>
v-else 基于表达式，对v-if补充 <h1 v-if="awesome">Vue is awesome!</h1> <h1 v-else>Oh no </h1>
v-else-if 类似
v-show 按条件显示一个元素（仅display） <h1 v-show="ok">Hello!</h1>
v-for 基于一个数组来渲染一个列表 const items = ref([{ message: 'Foo' }, { message: 'Bar' }])
<li v-for="item in items"> {{ item.message }} </li>
v-model 表单绑定 单行：<input v-model="text">
多行：<textarea v-model="message" placeholder="add multiple lines"></textarea>
单选：<input type="radio" v-model="pick" :value="first" /><input type="radio" v-model="pick" :value="second" />
复选：<input<br/> type="checkbox"<br/> v-model="toggle"<br/> true-value="yes"<br/> false-value="no" />
  • 同名简写：针对 v-bind的特殊简写，<div :id></div>or<div v-bind:id></div>
  • 动态绑定多个值：也是针对 v-bind的，通过不带参数的绑定<div v-bind="objectOfAttrs"></div>，这里的objectOfAttrs可能有多个属性值
  • JavaScript 表达式可以插入文本插值中，以及任何 Vue 指令attribute 的值中，并且仅支持表达式，不支持语句
  • 动态参数：指令参数上也可以使用一个 JavaScript 表达式，需要包含在一对方括号内，但是建议使用计算属性来替代。此外，内嵌模板尽量使用小写，防止浏览器强制转换。
  • 修饰符：以点开头的特殊后缀，表明指令需要以一些特殊的方式被绑定。例如 .prevent 修饰符会告知 v-on 指令对触发的事件调用 event.preventDefault()：<form @submit.prevent="onSubmit">...</form>
