vim

介绍vim的原因主要是这学期和wsl和linux等相关的实验较多，默认的编辑器也多是vim

vim是很受欢迎的一款编辑器，和传统的编辑器不同，它可以让我们的双手始终停留在键盘上，练习熟练后可以大大提高编程效率，在安装了wsl之后可以直接在终端打开vim，且可以通过个人喜好配置。

参考：https://zhuanlan.zhihu.com/p/150658147，https://www.bilibili.com/video/BV164411P7tw?spm_id_from=333.337.search-card.all.click&vd_source=caecf7af6ba7b03a92bb7722d0f97e1d

vim基本操作：https://zhuanlan.zhihu.com/p/68111471。

vim基础操作

vim有很多操作，光插入就好几种方式，这里我记录的是一些比较常用的。另外vim中的按键在不同模式下有不同用处。一般按esc确认自己是在normal模式，按：后表示输入指令。

打开文件

输入vim进入vim，输入vim<filename>可以通过vim去编辑文件，进入是普通模式，q是退出，wq是保存退出，(q!是不保存退出)，：saveas file 表示另存为。

另外如果要打开多个文件可以 vim <file1><file2>，默认会显示第一个文件

  • 输入：ls会打开文件序号，然后输入：b<序号例如1> 来切换到不同文件。
  • 如果要实现分屏操作，左右分屏可以 vim -On file1 file2，上下换成on。开启分屏可以：split左右分屏，：vsplit上下分屏。关闭分屏使用ctrl + w q。

文件编辑

在编辑模式下可以对文件进行修改，按esc键可以退回普通模式，如果不清楚自己在什么模式下也可以按esc回到normal模式，然后按v可以进入可视化模式。

插入

要进入插入模式有以下几种方法：

i 插入之前，a插入之后，I行首插入，A行尾插入，o下行插入，O上行插入

跳转

在normal模式下，hjkl表示左下上右

  • w快速跳转到单词开头
  • e快速跳转到单词结尾
  • b跳转到上一个单词的开头。
  • 跳转到行首是0，行尾是$
  • gg跳转到文件开头，G跳转到文件结尾，使用ctrl+o返回，ngg表示跳转到具体行。

查询

  • 按照字符搜索 f {char}
  • /time表示搜索time单词，按n表示下一个，*/#进行匹配。
  • ：set ic不区分大小写查询
  • ：set noic区分大小写
  • 另外一个搜索字符是 f<cha>，结合后面的内容可以表示删除到下一个a字母之前dfa,包括复制等

删除

快速删除：

  • x删除光标后的一个字符
  • dw删除后面的一个单词
  • diw表示删除光标所在的单词
  • daw删除前面的单词
  • dd是删除一整行
  • ndd删除n行
  • 另外d3->表示向右删除三个字符。
  • ：n1,n2d 表示删除两行号之间多行文本。

复制

yw复制单词，yy复制一行，p粘贴。

撤销

u撤销，ctrl r恢复

快速修改

  • r替换字符
  • s删除当前字符并且进入插入模式
  • cw可以在删除单词的基础上进入插入模式
  • 如果是引号内的内容可以输入ci"表示删除引号内的内容并重新填写。

重复

重复命令：n <command>多次进行

宏录制，指将多个操作进行一次命名录制，按下q表示开始录制，再按下<name>，然后记录键位后再正常模式下n@<name>即可自动进行n次操作。

常用命令

按下冒号之后可以输入一些命令指令,比如保存:w,保存并退出：:wq。

vim配置

cd回家之后，输入mkdir .vim新建文件夹，然后进入文件夹新建vimrc ：vim vimrc

noremap a b 输入a键达到b键的效果，b键本身效果也还在。

syntax on 打开高亮

set number 显示行号

set wildmenu命令补全

set hlsearch 搜索高亮

set ignorecase 忽略大小写

let mapleader=" " noremap <LEADER><CR> :nohlsearch<CR> 实现搜索高亮之后按空格和回车消除

为了更好的使用vim可以下载相关的插件，首先需要下载可以使用插件的一个vim-plug插件管理器，将文件放在.vim文件夹。

在vimrc中添加

然后在github上搜索一些插件，填入上面的代码后，输入命令：

  • PlugInstall 安装插件
  • PlugUpdate 更新或安装插件
  • PlugUpgrade 更新 vim-plug
  • PlugClean 移除不在列表中的插件
  • PlugStatus 查看插件状态

linux中复制文件：cp file.txt dir file2.txt dirl，最后的目录是文件夹
