公钥密码课程学习笔记

单向函数

公开密钥密码的设计关键：将单向函数构造为单向陷门函数

  1. 因数分解问题(Integer Factorization)

    给定大素数 p和q，求n=p×q , 只要一次乘法

    给定n，求p和q，即为因数分解问题IF，最快方法需要T(n)=exp(c((lnn)1/3ln(lnn))2/3)T(n)=exp(c((ln n)^{1/3} ln(ln n))^{2/3})T(n)=exp(c((lnn)1/3ln(lnn))2/3)次运算，其中c为大于1的正整数。

  2. 离散对数问题(Discrete Logarithm Problem)

    令素数p满足p-1含有另一大质数q (q整除p-1)，及一整数g, 1<g<p-1。

    若给定整数x，求y=g$^x $mod p是简单的

    但是若给定p, g及y，求x，则为DLP问题，是困难的。

  3. 背包问题(Knapsack Problem)

    给定有限个自然数序列集合B=(b1,b2,…bn)及二进制序列x=(x1,x2,…xn), xi∈(0,1), 求S＝∑xibi最多只需n-1次加法；

    但若给定B和S，求x则非常困难。

    穷举时有2n2^n2n种可能，当n很大时为计算上不可行。

RSA

基于大数分解困难，经典且最常见的公钥密码学算法

算法流程

①随机选择两个秘密大质数p和q

②计算公开模数n=p*q

③计算秘密的欧拉指数函数Φ(n)=(p-1)(q-1)

④选择一个与Φ(n)互素的数，作为e

⑤计算e模Φ(n)的乘法逆元素，作为d

  • 公开密钥PU={e,n}；私有密钥PR={d,n}

  • 加密：C=me(modn)C = m^e \pmod nC=me(modn)

  • 解密：m=Cd(modn)=med(modn)=mm = C^d \pmod n = m^{ed} \pmod n = mm=Cd(modn)=med(modn)=m

ElGamal 密码

  • 公共模数素数P，本原元素α，私钥x，公钥Y=αxα^xαx mod P

  • 加密，消息1≤m≤P-1：

    • A随机选择k∈(1, P-1)
    • A访问公共区域获取B的公钥YB = αxBα^{x_B}αxB​ mod P。计算：
    • K = (YB)k(Y_B)^k(YB​)k mod P
    • c1 = $α^k $mod P，c2 = mK mod P
    • 密文即为 (c1, c2)

  • 解密：

    • B首先恢复K：K=c1xBmodP=αkxBmodPK = c_1^{x_B} mod P = α^{kx_B} mod PK=c1xB​​modP=αkxB​modP
    • 然后恢复m：m=c2/KmodP=c2K−1modPm = c_2/K mod P = c_2K^{-1} mod Pm=c2​/KmodP=c2​K−1modP

ElGamal密码体制：

  • 是概率密码体制，相同明文每次加密得到不同的密文
  • 加密效率是50％，因为密文大小是明文的两倍椭圆曲线密码算法
  • K不能重复使用，否则直接m1/m2=c2,1/c2,2

ECC椭圆曲线密码

基于椭圆曲线离散对数难题，现有系统广泛使用，基本上已经替代RSA

和RSA对比：

  • ECC中的加法运算与RSA中的乘法运算相对应
  • ECC中的数乘运算与RSA中的模幂运算相对应
  • 同等密码强度下，ECC所需密钥量和计算量都小于RSA算法
  • 同等密钥长度下，ECC计算量与RSA相当

椭圆曲线定义

  • 曲线：常见为素域Weierstrass曲线：y2=x3+ax+by^2=x^3+ax+by2=x3+ax+b

    • 表示为E(a,b)，a和 b 是曲线的参数，满足一定的条件以保证曲线不包含奇异点

    • 域标识GF§，曲线通常定义在有限域 FpF_pFp​或 F2mF_{2^m}F2m​，其中 p 是质数，或者是二次扩展域

    • 无穷远点：单位元/零元

    • 基准点(base)G：指定的椭圆曲线上的一点（公开）

    • 阶： G点的阶N，使得：NG=O

  • 操作： 定义“点加Point addition”和“倍点Point Doubling“操作

ECC的基本操作

  1. 点加（Point Addition）：

    假设有两个点P(x1,y1)∈E(Fp)\{O}P(x_1, y_1)\in E(F_p)\backslash\lbrace O\rbraceP(x1​,y1​)∈E(Fp​)\{O}和Q(x2,y2)Q(x_2, y_2)Q(x2​,y2​)在椭圆曲线上, P≠Q（x1≠x2x_1\neq x_2x1​​=x2​），它们的和 R=P+Q 的计算公式如下：

    λ=y2−y1x2−x1(modp)\lambda = \frac{y_2 - y_1}{x_2 - x_1} \pmod{p} λ=x2​−x1​y2​−y1​​(modp)

    x3=λ2−x1−x2(modp)x_3 = \lambda^2 - x_1 - x_2 \pmod{p} x3​=λ2−x1​−x2​(modp)

    y3=λ(x1−x3)−y1(modp)y_3 = \lambda(x_1 - x_3) - y_1 \pmod{p} y3​=λ(x1​−x3​)−y1​(modp)

  2. 倍点(Point Doubling)：

    λ=3x12+a2y1(modp)\lambda = \frac{3x_1^2 + a}{2y_1} \pmod{p} λ=2y1​3x12​+a​(modp)

    x3=λ2−2x1(modp)x_3 = \lambda^2 - 2x_1 \pmod{p} x3​=λ2−2x1​(modp)

    y3=λ(x1−x3)−y1(modp)y_3 = \lambda(x_1 - x_3) - y_1 \pmod{p} y3​=λ(x1​−x3​)−y1​(modp)

这两个公式就是简单的数学求导

  1. 标量乘（scalar multiplication） \ 多倍点（Point Multiplication）

    计算Q=kP，即椭圆曲线加法的反复执行

    椭圆曲线离散对数难题：当Q已知时计算k，“Pollard rho方法”是目前最快的方法

公钥系统

  • 选择正整数e作为私有密钥
  • 公开密钥为Q=eG

介绍一些椭圆曲线上实现的密码算法，幂运算对应点乘

Menozes-Vanstone椭圆曲线密码

以Menozes-Vanstone为例介绍加解密

  1. 令E是GF§上的椭圆曲线，选取基点G，阶为N
  2. 私钥：e<N
  3. 公钥：E，GF§，Q=eG
  4. 加密：
    • 明文m=(x1,x2)，x1,x2<p
    • 任取v<N，vQ=(c1,c2)
    • 密文(y0,y1,y2)，y0=vG, y1=c1x1 mod p, y2=c2x2 mod p
  5. 解密：
  • 用私钥e计算：ey0=evG=vQ=(c1,c2)
  • m=(x1,x2)=(y1c1−1^{-1}−1mod p, y2c2−1^{-1}−1 mod p)

ElGamal算法在椭圆曲线上的实现

DH，将幂运算更改为曲线上的点乘运算

  1. E(a,b)，基点G∈E
  2. B选择a并保密, 0<a<N，N为G的阶(order) ，公钥为aG
  3. A向B发送消息m
    • A将m嵌入点Pm，选择随机数k（私钥），对应共享信息是k(aG)，发送(kG, Pm +k(aG))
    • B计算Pm = (Pm +k(aG)) –a(kG)

消息编码

将消息映射到Pm的一种方法：

  • 编码：
    • 将域p划分为长256的小段
    • 对明文进行分组：使得每个分组0 ≤ m ≤ $\lfloor p/256\rfloor $
    • 对明文分组进行编码，使之成为由域参数给出的椭圆曲线上的点Pm
      • 在256m≤x<256(m+1)中找到一个x，使得椭圆曲线方程有解
      • 一般地，对所有的满足256m≤x<256(m+l)的x，椭圆曲线方程都无解的概率是很小的。从而可以完成编码。
  • 解码：
    • 若解密横坐标落在256m≤x<256(m+1)中，则解码为m

Massey-Omura公钥体制

不需要交换公钥

GF(q)上：

  • 用户A 加密、解密密钥：eA, dA
    • gcd(eA,q-1)=1, eA dA =1 mod (q-1)
  • 用户B 加密、解密密钥：eB, dB
    • gcd(eB,q-1)=1, eB dB =1 mod (q-1)

流程

  1. A->B：meAm^{e_A}meA​
  2. B->A：meAeBm^{e_Ae_B}meA​eB​
  3. A->B：(meAeB)dA=meB(m^{e_Ae_B})^{d_A}=m^{e_B}(meA​eB​)dA​=meB​
  4. B：(meB)dB=m(m^{e_B})^{d_B}=m(meB​)dB​=m

E(a,b)上：

  • m编码为椭圆曲线上的点Pm
  • N: 椭圆曲线上的点数
  • 用户随机选择e: 1<e<N, gcd(e, N)=1, ed=1 mod N

流程

  1. A->B：eAPme_A P_meA​Pm​
  2. B->A：eBeAPme_B e_A P_meB​eA​Pm​
  3. A->B：dA(eBeAPm)=eBPmd_A(e_Be_AP_m)=e_BP_mdA​(eB​eA​Pm​)=eB​Pm​
  4. B：dB(eBPm)=Pmd_B(e_BP_m)=P_mdB​(eB​Pm​)=Pm​

Rabin密码

虽然普遍认为RSA破解和大数分解同等困难，但并没有严格证明，Rabin是第一个严格证明和大数分解同等困难的公钥方案

密钥生成：

1.随机生成两个不同但大小相近的大素数p和q

2.计算n=pq （p和q模4余3时，利于计算）

3.公钥为n，私钥为(p,q)

加密：B为A加密消息

1.将消息分组表示成{0,1,…,n-1}中的整数m，n是A的公钥

2.计算c=m2^22 mod n作为密文

解密：

1.求解c的四个平方根

从四个平方根中选取正确的一个

背包公钥加密

基于子集和问题：选择一个容易求解的子集和问题实例（私钥），然后将它伪装成一个很难求解的一般子集和问题实例（公钥）

Merkle-Hellman背包加密方案

第一个具体实现的，但已有一个攻击它的多项式时间算法

超递增序列

超递增序列(b1,b2,…,bn)是一个正整数序列，其中每个元素都大于前面所有元素之和

  • 求解超递增序列子集和问题
  • 输入(b1,b2,…,bn)，整数s
  • 输出(x1,x2,…,xn)，其中xi=1表示选中bi
  • i=n
  • 当i ≥ 1时，执行
    • 若s ≥ bi，则xi=1, s=s - bi 。否则xi=0
    • i=i-1
  • 返回(x1,x2,…,xn)

加解密流程

  • 公/私钥生成：

    公共参数n

    1.选择一个超递增序列(b1,b2,…,bn)和模数M，使得M>sum(bi)

    2.随机选择一个整数W，1 ≤ W ≤ M-1，gcd(W,M)=1

    3.随机选择整数{1,2,…,n}的一个置换π

    4.计算ai=Wbπ(i)_{π(i)}π(i)​ mod M, i=1,2,…,n

    5.公钥为(a1,a2,…,an) ，私钥为(π,M,W, (b1,b2,…,bn))

  • 加密：消息m

    1.将消息表示为二进制串，m=m1,m2,…,mn

    2.计算c=m1a1+m2a2+…+mnan，作为密文

  • 解密：

    1.计算d=W−1^{-1}−1c mod M

    2.求满足d=r1b1+r2b2+…+rnbn的二进制数r1,r2,…,rn

    3.消息比特是mi=rπ(i)_{π(i)}π(i)​, i=1,2,…,n

概率公钥密码

概率密码利用随机性得到一个可证的高强度安全性

确定性的密码体制存在缺点：

  • 密码体制对消息空间的所有概率分布未必是安全的
    • 例如：RSA中，消息0或1总是加密成其本身
  • 同一消息发送两次时，密文重复，容易被检测
  • 有利于密码分析

在对称密码体制中实现概率密码:

①在消息中嵌入随机数

②通过可逆变换使得该随机数影响到所有比特位

③加密

Blum-Goldwasser概率加密

已知的最有效的概率加密方案：

  • 在速度和消息扩展方面可以与RSA相比
  • 若大数分解是困难的，则它是理想安全的
  • 易受选择密文攻击，在实际应用中受限

密钥生成：

  • 随机选择两个大素数p,q，p ≡ q ≡ 3 mod 4
  • 计算n=pq
  • 用扩展欧几里得算法计算整数a和b，使得ap+bq=1
  • 公钥是n，私钥是(p,q,a,b)

**加密：**n为接收方公钥

  1. 令k=⌊log2n⌋\lfloor log_2n\rfloor⌊log2​n⌋, h=⌊log2k⌋\lfloor log_2k\rfloor⌊log2​k⌋，将消息表示为长度为t的串m=m1m_1m1​m2m_2m2​…mtm_tmt​，其中mi是长度为h的二进制串
    用BBS生成器产生密钥流加密m

  2. 随机选择r，并计算x0x_0x0​=r2^22 mod n

  3. 对i=1…t执行：

    a)计算xix_ixi​=xix_ixi​-12 mod n

    b)记pip_ipi​为xix_ixi​的最低h位比特

    c)计算cic_ici​=pip_ipi​⊕⊕⊕m_i$

  4. 计算xt+1_{t+1}t+1​=xt2x_t^2xt2​ mod n

  5. 发送密文c = (c1,c2,…,ct,xt+1_{t+1}t+1​)

解密：

1.计算d1d_1d1​=((p+1)/4)t+1^{t+1}t+1 mod (p-1)

2.计算d2d_2d2​=((q+1)/4)t+1^{t+1}t+1mod (q-1)

3.计算u=xt+1_{t+1}t+1​d1^{d_1}d1​ mod p, v=xt+1_{t+1}t+1​d1^{d_1}d1​ mod q

4.计算x0x_0x0​=vap+ubq mod n

5.对于i=1…t，执行

a)计算xix_ixi​=xi−1x_{i-1}xi−1​2^22 mod n

b)记pip_ipi​为xix_ixi​的最低h位比特

c)计算mim_imi​=pip_ipi​⊕cic_ici​

SM2椭圆曲线公钥密码*

我国密码管理局2010年颁布的自主知识产权的商用密码算法，是椭圆曲线密码与序列密码的组合

用椭圆曲线密码得到一个随机的秘密的点，作为序列密码的种子

用hash函数在CRT模式下，产生伪随机序列作为加密密钥流

系统参数：

  • 在GF§中选取椭圆曲线 E：y2=x3+ax+by^2=x^3+ax+by2=x3+ax+b，p是大素数，(4a3+27b24a^3+27b^24a3+27b2) mod p≠0
  • 选取基点G，G的阶n是一个大素数

密钥扩展函数 KDF(Z, klen)：

  • 输入：比特串Z，整数 klen

  • 输出：长为 klen 的比特串

  1. 初始化计数器 ct=0x00000001

  2. i从1到 ⌈Klen/v⌉ 执行

    ①计算 Hi=hash(Z||ct)，其中hash()是输出v比特的散列函数

    ②ct=ct+1

  3. 输出 H1||H2||…。若总长度超过Klen时，最后一个分组只输出左边若干个比特

加解密流程

  • 私钥：1<d<n-1
  • 公钥：P=dG
    • S=hP， h为余因子，是椭圆曲线上所有点的个数与n相除的整数部分
    • 若S是无穷远点，则需要重新设置私钥

  1. **加密算法：**消息为m，len为m的长度

    1.随机选取 k，1<k<n-1

    2.C1=kG=(x1, y1)

    3.R=kP=(x2, y2) =kdG

    4.t=KDF(x2||y2, len)。若 t 为全0比特串，则返回第1步重新选取k

    5.c2=m⊕t

    6.c3=hash(x2||m||y2)

    7.输出密文 c=C1||c2||c3

  2. 解密算法: len为密文中 c2 的长度

    1. 验证C1是椭圆曲线上的点。若不满足，则报错退出
    2. S=hC1，若S是无穷远点，则报错退出
    3. R=dC1=(x2, y2) =dkG
    4. t=KDF(x2||y2, len)。若 t 为全0比特串，则报错退出
    5. m=c2⊕t
    6. u=hash(x2||m||y2)，若u≠c3，则报错退出
    7. 输出明文m
